# 📊 台股專業分析系統 

> 一個功能完整、效能優化的台股技術分析工具
> 
## 🎯 功能特色

- ✅ 完整技術分析（MA, RSI, KD, MACD, 布林通道）
- ✅ 自選股管理（LocalStorage 儲存）
- ✅ 智能警示系統
- ✅ AI 分析報告
- ✅ 響應式設計（支援手機/平板/電腦）

## 🚀 快速開始

### 直接開啟
1. 下載 `taiwan_stock_analyzer.html`
2. 用瀏覽器開啟
3. 開始使用！

## 📖 使用說明

### 查詢股票
1. 選擇市場（上市/上櫃）
2. 輸入股票代號（例如：2330）
3. 選擇日期範圍
4. 點擊「開始分析」

### 自選股管理
- 加入：選擇市場 → 輸入代號 → 點擊「加入自選」
- 更新：點擊「更新報價」按鈕
- 移除：點擊標籤上的「✕」
